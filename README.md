A portable environmental monitoring device combining DHT11 sensor, ESP32 microcontroller, and 0.91" OLED display in a compact 110×55×28mm matte-black enclosure. Powered by 18650 battery with USB charging, it provides real-time temperature and humidity readings with wireless connectivity. Suitable for HVAC systems, agriculture, IoT applications, and environmental monitoring requiring accurate, portable sensing solutions with user-friendly interface.
